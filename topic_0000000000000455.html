cgMoveGenerator We generate all possible moves(regardless of blocking) for all pieces once here. So this class is used once at the very beginning of the very first game. This technique saves a huge amount of computation for the Engine, as it only has to look up these moves and test their legality on a given board - instead of actually generating them.
cgMoveSet Stores information about all possible moves a piece can make on a given index position on the board.
